# IQ Test Challenge

## Overview

IQ Test Challenge is a modern, interactive web-based quiz application designed to evaluate logical reasoning, pattern recognition, numerical aptitude, and problem-solving skills through a series of IQ-style questions.

The application provides a clean and engaging user interface with real-time scoring, countdown timers, animated visual elements, progress tracking, and detailed answer reviews.

---

## Features

* Interactive multiple-choice IQ questions
* Real-time countdown timer for each question
* Automatic answer validation
* Progress tracking bar
* Animated user interface
* IQ score calculation
* Performance level assessment
* Review section for incorrect answers
* Responsive design for desktop and mobile devices
* Smooth animations and transitions

## How It Works

1. The user starts the quiz.
2. Questions are displayed one at a time.
3. A 30-second timer is provided for each question.
4. Selecting an answer immediately checks correctness.
5. Correct answers increase the score.
6. After all questions are completed, the system:

   * Calculates the IQ score
   * Displays performance level
   * Shows explanations for incorrect answers
